Steps to create an ISO document such as a PDTS from an HTML working draft using LibreOffice Writer:

Create the initial .odt file

  • Open the HTML file in Writer.
  • View > Print Layout
  • File > Export...
    • Save as type: OpenOffice.org Text (Writer/Web)(.odt)(*.odt)

    • File name: name.odf

    • Save
    • Close Writer

  • Git Add, Commit name.odt

First edit pass

Open name.odt with Writer. Do a pass through the document:

  • Add Foreword boilerplate.

  • Remove grayed italic editorial comments.

  • Set up the Normative references [fs.norm.ref] footnote: Insert > Footnote/Endnote.

  • Fix formatting boo boo's, such as table widths. (Try Table > Autofit, etc)
