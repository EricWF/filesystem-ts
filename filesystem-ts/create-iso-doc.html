Steps to create an ISO document such as a PDTS from an HTML working draft using LibreOffice Writer:

Create the initial .odt file

  • Open the HTML file in Writer.
  • View > Print Layout
  • File > Export...
    • Save as type: OpenOffice.org Text (Writer/Web)(.odt)(*.odt)

    • File name: name.odf

    • Save
    • Close Writer

  • Git Add, Commit name.odt

First edit pass

  • Add Title and Foreword boilerplate - but without page breaks.

  • Create the Preface style. LibreOffice 4.0 Writer Guide page 144

  • Set the page styles:

    • Title page - right click on status bar, 3rd box from left, select: First Page

    • Set cursor on first frontmatter text - Insert > Manual Break > Page break, style: Preface, Change page number, 1.

    • Set cursor on first body pages text - Insert > Manual Break > Page break, style: Default Style, Change page number, 1.

  • For each of the three page styles, cursor to appropriate page, Format > Page... > Page tab > Format: A4

  • Close and commit.

Second edit pass

Open name.odt with Writer. Do a pass through the document:

  • Remove grayed italic editorial comments.

  • Set up the Normative references [fs.norm.ref] footnote: Insert > Footnote/Endnote.

  • Fix formatting boo boo's, such as table widths. (Try Table > Autofit, etc)

  • Close and commit.
