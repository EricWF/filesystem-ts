Steps to create an ISO document such as a PDTS from an HTML working draft using LibreOffice Writer:

Create the initial .odt file

The conversion goes through an intermediate .rtf file to strip out the HTML page style and replace it with Default Style. There must be an easier way to do this, but neither I nor Google could find it!

  • Open the HTML file in Writer.
  • File > Export...
    • Save as type: OpenOffice.org Text (Writer/Web)(.odt)(*.odt)

    • File name: temp.odt

    • Save
    • Close Writer
  • Open temp.odt with Writer
    • File > Save As...
    • Save as type: Rich Text (.rtf)(*.rtf)

    • File name: temp.rtf.

    • Confirm File Format: Use Rich Text Format.

    • Close Writer.

  • Open temp.rtf with Writer
    • File > Save As...
    • Save as type: ODF Text Documdent (.odt)(*.odt)

    • File name: name.odf.

    • Save

    • Close Writer.

  • Git Add, Commit name.odt
